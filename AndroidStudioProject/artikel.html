PERKULIAHAN DI SEMESTER PERTAMA

Hello!

Perkenalkan namaku Hanum Fauziah Isnanto, mahasiswa baru Teknik Informatika, Universitas Dian Nuswantoro Semarang. Tahun ini merupakan semester pertama dalam perkuliahan yang aku jalani, dalam semester awal ini terdapat 7 mata kuliah yang di pelajari, salah satu mata kuliah yang aku pelajari adalah dasar komputasi, dimana pada kali ini mahasiswa berkesempatan untuk membuat suatu halaman website. Dalam pembuatan website ini menggunakan HTML. Pada awalnya aku belum tau apa itu HTML, bahkan kata HTML saja terasa asing, setelah mendapat materi tentang HTML pada perkuliahan kali ini dan mencoba membuat suatu halaman website, ternyata belajar HTML sangat menyenangkan.

HTML merupakan kepanjnagan dari Hypertext Markup Language, HTML sendiri mendeskripsikan struktur dari halaman web menggunakan markup sedangkan elemen-elemen dalam HTML merupakan blok-blok yang membangun halaman HTML. elemen-elemen dalam HTML dipresentasikan melalui tags. Dalam pembuatan halaman HTML ada beberapa editor yang dapat digunakan antara lain notepad, notepad++, sublime, visual studio code,dll. Kali ini aku menggunakan editor VScode dalam pembuatan halaman HTML. Didalam struktur HTML terdapat head dan body sebagai komponen utama. Didalam tags head biasanya berisi title(berfungsi memberi judul pada halaman suatu website). Sedangkan pada tags body dapat disisipkan berupa gambar, vidio maupun link. Pada bagian inilah ang ditampilkan pada halaman web.
